Loan Application Form

Personal Information

Full Name is required (2-50 characters).
Date of Birth is required.
Gender is required.
Marital Status is required.
Valid Email Address is required.
Valid 10-digit Phone Number is required.
Street Address is required (5-100 characters).
City is required (2-50 characters).
State is required (2-50 characters).
Valid ZIP Code is required (5-6 digits).
Valid PAN Number is required (e.g., ABCDE1234F).
Valid 12-digit Aadhaar Number is required.

Employment Details

Employment Status is required.
Employer Name is required (2-50 characters).
Employer Street Address is required (5-100 characters).
City is required (2-50 characters).
State is required (2-50 characters).
Valid ZIP Code is required (5-6 digits).
Job Title is required (2-50 characters).
Annual Income is required and must be a positive number.
Work Experience is required and must be a non-negative number.

Loan Details

Loan Amount is required and must be a positive number.
Loan Purpose is required.
Loan Tenure is required and must be between 1 and 30 years.

Financial Details

Bank Name is required (2-50 characters).
Valid Account Number is required (9-18 digits).
Valid IFSC Code is required (e.g., ABCD0123456).
